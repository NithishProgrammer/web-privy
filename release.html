Release Update

We are excited to announce a major update to Privilayer.

Privilayer has always helped developers protect sensitive text before sending prompts to Large Language Models. Now we're expanding beyond text into the visual world.

The new release introduces Image-based PII Detection & Automatic Masking.

Image-to-Image Masking

Upload an image and Privilayer automatically detects faces, documents, ID cards, license plates, and other sensitive information, returning a fully masked image ready for AI.

Optimized Performance

Designed with ultra-low latency in mind, enabling high-speed image processing without slowing down your LLM workflow.

High Accuracy

Advanced detection models minimize false positives while ensuring sensitive information never slips through.

How it works: Core Workflow & Lifecycle

Input: The system locally intercepts raw visual data—such as image uploads or document scans—before any transmission occurs.

Targeted Detect: Advanced computer vision scans the input to specifically identify highly sensitive identity documents, including Aadhaar, PAN, Passports, and other documents.

Masks: PrivyLayer applies permanent black box overlays directly onto the detected confidential information, completely obscuring the sensitive pixels before the image is sent out.

Images are Irreversible: The visual sanitization process is strictly a one-way pipeline. Once the image is processed for the AI, the original visual state cannot be reconstructed or reversed.

Requirements

To maintain low-latency localized processing and handle real-time computer vision workloads, the host system must meet the following infrastructure criteria:

GPU Acceleration: Requires dedicated graphics hardware optimized for parallel image processing and computer vision workloads.

Minimum Requirement: An NVIDIA T4 GPU (or equivalent) to ensure better, real-time document detection and instant black-box masking before data transmission.

Install it here (Developers)

pip install privylayer-images

Example Usage

import privyimage

img_path = "example.jpg"

privyimage.mask_img(img_path)

# The output image will be "path/to/image_masked.jpg" with all sensitive information masked.
